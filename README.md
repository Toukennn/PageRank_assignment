# PageRank Algorithm

This repository contains a Python implementation of the PageRank algorithm
developed as part of a university assignment.

The project focuses on computing PageRank scores using an iterative method
and includes handling of dangling nodes.

---

## Repository Structure

- `notebook/`  
  Contains the google colab notebook with the full implementation.

- `assignment/`  
  Contains the original assignment description (PDF).

---

## How to Run

The notebook can be executed using:
- Google Colab
- Local Jupyter Notebook environment

No additional setup is required beyond standard Python scientific libraries.
